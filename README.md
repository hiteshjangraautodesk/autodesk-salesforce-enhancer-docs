# Public web pages (Chrome Web Store)

Upload the contents of this folder to **Autodesk-Salesforce-Enhancer-Public-Docs** and enable GitHub Pages.

## Files

| File | Chrome Web Store field |
|------|------------------------|
| `home.html` | **Homepage URL** |
| `support.html` | **Support URL** |
| `privacy.html` | **Privacy policy URL** |
| `styles.css` | Shared styling (keep alongside HTML) |

## GitHub Pages URLs

| Page | URL |
|------|-----|
| Home | `https://hiteshjangraautodesk.github.io/Autodesk-Salesforce-Enhancer-Public-Docs/home.html` |
| Support | `https://hiteshjangraautodesk.github.io/Autodesk-Salesforce-Enhancer-Public-Docs/support.html` |
| Privacy | `https://hiteshjangraautodesk.github.io/Autodesk-Salesforce-Enhancer-Public-Docs/privacy.html` |

## Setup

1. Copy all files in `store/public/` to the docs repo root.
2. Enable **Settings → Pages** on the docs repo.
3. Paste the three URLs above into the Chrome Web Store dashboard.
